Note that -c, or -classname is an option. Otherwise the files will be placed in the directory correlating to the input filename.

#############################################################################################
Command to Run build-terraform:  ############################################################
#############################################################################################

	python build-terraform.py -f <input filename>

		or

	python build-terraform.py -f <input filename> -c <class name>


#############################################################################################
Examples:  ##################################################################################
#############################################################################################

	python build-terraform.py -f Comp880.M1-Summer2017-input.dat

	python build-terraform.py -f Comp880.M1-Summer2017-input.dat -c COMP880.M1-Summer2017
